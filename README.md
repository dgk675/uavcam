# UAVCam

## Setup
1. Flash a Micro SD card with Raspberry Pi OS and boot it up.
2. Update the image using
    ```sh
    sudo apt-get update
    sudo apt-get upgrade -y
    ```
3. Install python dependencies
    ```sh
    pip install loguru tomli yamspy
    ```
